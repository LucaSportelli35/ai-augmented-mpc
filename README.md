# AI-Augmented MPC Project Website

This folder is ready for GitHub Pages.

## Files
- index.html: webpage content
- style.css: visual style
- assets/: figures used in the page

## Quick deployment with GitHub Pages
1. Create a public GitHub repository named: ai-augmented-mpc
2. Upload all files in this folder to the repository root.
3. Go to Settings > Pages.
4. Under "Build and deployment", choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. Wait 1-3 minutes.
7. Your page will be available at:
   https://YOUR_USERNAME.github.io/ai-augmented-mpc/
