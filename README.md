## Update instructions

To update the GitHub Pages website:

1. Go to the GitHub repository.
2. Click **Add file → Upload files**.
3. Upload/replace:
   - `index.html`
   - `style.css`
   - `README.md`
   - `assets/`
4. Click **Commit changes**.
5. Wait around 1–3 minutes.
6. Refresh the website.
