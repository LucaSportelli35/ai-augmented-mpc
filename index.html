Project Overview

Autonomous rendezvous and proximity operations in adversarial orbital environments require guidance systems capable of balancing target pursuit, safety preservation, and real-time adaptability under dynamically evolving interaction conditions.

This work presents an adaptive Model Predictive Control framework for autonomous spacecraft RPO in a multi-agent adversarial scenario. The key idea is that the learning layer does not directly command thrust. Instead, it modifies high-level, physically interpretable MPC parameters such as tracking weights, prediction horizons, safety penalties, and keep-out-zone objectives.

Method

Constrained MPC inner loop

At each control update, the controller solves a finite-horizon quadratic program using Clohessy-Wiltshire relative orbital dynamics, actuator bounds, predictive safety constraints, and feasibility-preserving slack variables.

Adaptive supervisory outer loop

The adaptive layer changes optimization-level parameters online, including prediction horizons, control horizons, tracking weights, safety penalties, and minimum separation objectives.

Interpretability by design

Since adaptation acts on meaningful MPC quantities rather than raw thrust commands, the resulting behavior can be analyzed through physical geometry, safety metrics, and parameter trends.

Parametric constrained MPC problem

$$z^\star(x_k,P_k)=\arg\min_z \frac{1}{2}z^\top H(P_k)z+f(x_k,P_k)^\top z$$ $$\text{s.t.}\quad A(x_k,P_k)z\le b(x_k,P_k),\qquad z_{lb}\le z\le z_{ub}$$

MPC prediction horizon and predictive safety-constraint evaluation. The optimizer anticipates Bandit and Guard motion and adjusts the predicted trajectory when future keep-out-zone risk becomes active.

Adaptive Parameters and Interpretability

Because the supervisory layer acts on interpretable MPC parameters, the adaptive behavior can be analyzed through quantities such as Guard safety weight, minimum separation objective, target distance, and Guard proximity.
