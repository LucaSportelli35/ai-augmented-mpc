Abstract

Autonomous rendezvous and proximity operations in adversarial orbital environments require guidance architectures that balance target pursuit, safety preservation, and real-time adaptability. This project presents an adaptive Model Predictive Control framework for spacecraft rendezvous and proximity operations in a multi-agent adversarial scenario.

The key design choice is that the learning layer does not directly command thrust. Instead, it updates high-level and physically interpretable MPC parameters, while the constrained optimization problem remains responsible for generating thrust commands under orbital dynamics, actuator limits, predictive keep-out-zone constraints, and feasibility-preserving slack variables.

Method

Constrained MPC inner loop

The inner loop solves a finite-horizon quadratic program at every control update using Clohessy-Wiltshire relative orbital dynamics. The optimizer tracks the target spacecraft while enforcing actuator limits and predictive safety constraints around the adversarial spacecraft.

Adaptive supervisory outer loop

The outer loop updates interpretable MPC parameters online, including prediction horizon, control horizon, terminal weights, safety penalty, and minimum separation objective. This creates adaptive behavior without replacing the constrained MPC structure.
